My Projects

Meme in a Giffy

A React app created with Danielle Alamilla, Camila Aparicio, and Dre Nnamdi. Using the GIPHY API, users can search for gifs, add their own text caption, tag(s), and save it all together as a meme to Firebase. Users can then go to the Browse section and look at saved memes, search for ones with specific caption/tags, and vote on them.

To Do List

A React to-do list, users can add or remove as many tasks as needed, then save their list to Firebase. The saved list will then be display in the below section.

Users can also interact with the saved list. They can edit task(s), cross out ones they've finished, and delete the list if they've finished all tasks (or like to pretend they have)

Youtube Stats Collector

Using the Youtube API, this Python app collects the statistics of every video in a Youtube playlist, or every video uploaded by a channel. The result is then outputted in a .csv file. This app comes with a GUI, user input handling and error handling features to make the user experience as straighforward, hassle free as possible. To see what can be done with the data collected by this app, check out my "Other Projects" below.
