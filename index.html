Featured Projects

Meme in a Giffy

A primitive React meme app created with Camila Aparicio, Danielle Alamilla and Dre Nnamdi. Utilizing the GIPHY API, users can search for gifs, add their own caption, tag(s), and save it all together as a meme to Firebase.

Users can then browse saved memes, search for ones with specific caption/tags, and vote on them.

Youtube Stats Collector

Using the Youtube API, retrieve quantitative data of all videos from a playlist or uploaded by a channel as specified by the user given URL. Outputs the retrieved data into a .CSV file.

My proudest project yet, since I worked on this by myself (with the help of Google and Stackoverflow of course). It even has some crude redundancy features to help users avoid simple errors. I even made it into a GUI with "tkinter", rather than leaving it as a console script.
