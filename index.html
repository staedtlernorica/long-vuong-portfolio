Projects & Demos

Here you can find my projects, ordered stochastically.

Trivia Quiz Webpage

Retrieve trivia questions from a trivia database API that matches user's input settings, and run the questions through a trivia quiz UI (think "Who Wants to be a Millionaire").

Running Calculator

Given two of time/distance/pace, will calculate the remaining quantity. Seamlessly switch between metric and imperial units at the click of a button, rather than having to reset the calculator after every new input.

Youtube Playlist/Channel Statistics Scraper

Using the Youtube API, retrieve quantitative data of all videos from a playlist/uploaded by a channel as specified by the user given URL. Outputs the retrieved data into a .CSV file.

My proudest project yet, since I worked on this by myself (with the help of Google and Stackoverflow of course). It even has some crude redundancy features to help users avoid simple errors. I even made it into a GUI with "tkinter", rather than leaving it as a console script.
